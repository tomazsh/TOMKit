addButtonWithTitle:block:

Adds a custom button to the action sheet with a block to be executed on click.

- (NSInteger)addButtonWithTitle:(NSString *)title block:(NNNActionSheetButtonBlock)block

Parameters

title

The title of the new button.

block

The block to be executed on button click. This block takes two parameters: the current action sheet and index of the button.

Return Value

The index of the new button. Button indices start at 0 and increase in the order they are added.
